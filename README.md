# Pretest Recruitment - REST API Jadwal Pelajaran Sekolah

REST API untuk mengelola jadwal pelajaran sekolah.

## Fitur

- CRUD Jadwal Pelajaran
- Autentikasi API Key
- Upload Excel Jadwal
- Export Excel Rekap JP
- Endpoint Siswa
- Endpoint Guru
- Endpoint Yayasan/Admin
- Deteksi bentrok jadwal kelas/guru

## Teknologi

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- Multer
- ExcelJS
- XLSX
